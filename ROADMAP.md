# Roadmap / Дорожная карта

## 0.1 public beta / Публичная бета 0.1

- local-first two-site workspace;
- portable site manifests;
- ChatGPT application briefs;
- multilingual public product site;
- Cloudflare Worker deployment;
- public governance and source-available licensing.

- локальная рабочая область для двух сайтов;
- переносимые манифесты;
- задания для приложения ChatGPT;
- многоязычный публичный сайт продукта;
- развёртывание Cloudflare Worker;
- публичное управление и source-available лицензирование.

## 0.2 / Версия 0.2

- repository template generation;
- downloadable Cloudflare starter bundles;
- manifest schema validation and migration;
- accessible guided onboarding;
- opt-in encrypted backup design.

- генерация шаблона репозитория;
- загружаемые стартовые комплекты Cloudflare;
- проверка и миграция схемы манифеста;
- доступный пошаговый onboarding;
- проект добровольного зашифрованного резервного копирования.

Creator and Studio plans remain non-operational until billing, identity, tax, privacy, consumer-contract and support processes receive qualified Italian and European review.

Тарифы Creator и Studio не запускаются до квалифицированной итальянской и европейской проверки оплаты, идентификации, налогов, конфиденциальности, потребительских договоров и процессов поддержки.
