# Roadmap / Дорожная карта

## 0.2 voice-first beta / Voice-first beta 0.2

- canonical locale paths and flag-based language navigation;
- standard CLARYEL beta banner and brand mark;
- voice-generated website briefs and continuous change requests;
- design-review instructions for AI development;
- two active sites per free Account Holder;
- protected Cloudflare deployment.

- канонические языковые пути и переключатель с флагами;
- стандартная beta-плашка и знак CLARYEL;
- голосовые задания на создание сайта и постоянные изменения;
- инструкции по оценке дизайна для ИИ-разработчика;
- два активных сайта на бесплатного владельца аккаунта;
- защищённая публикация Cloudflare.

## Planned / Планируется

- direct GitHub repository onboarding with explicit user approval;
- encrypted account backup and cross-device project history;
- visual before/after review for proposed design changes;
- direct deployment status in the Community workspace;
- Creator and Studio account plans after legal and billing review.

- прямое подключение GitHub-репозитория с явным подтверждением пользователя;
- зашифрованное резервное копирование аккаунта и история между устройствами;
- визуальное сравнение до/после для предлагаемых изменений дизайна;
- статус публикации непосредственно в Community workspace;
- тарифы Creator и Studio после юридической и платёжной проверки.
